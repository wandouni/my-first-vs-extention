# antd-code-helper

This is the README for your extension "antd-code-helper".

## Features

根据 model 类型定义，快捷生成 antd 表格 columns 数组代码。

- 支持两种类型定义格式
- 支持复制到剪贴板
- 支持插入到当前光标处

## Requirements

没有任何依赖，直接可用
